= Get the DataStax Apache Pulsar(TM) Connector configuration
:navtitle: Get the connector configuration

Use the `pulsar-admin sinks get` command to retrieve detailed information on a Pulsar sink:

[source,bash]
----
bin/pulsar-admin sinks get --name sink-name
----

Replace `sink-name` with the name of the sink associated with your DataStax Pulsar Connector.

For example, the following command retrieves the configuration for a sink named `dse-sink-kv`:

[source,bash]
----
bin/pulsar-admin sinks get --name dse-sink-kv
----

The configuration is returned in JSON format.
