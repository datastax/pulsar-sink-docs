= Update the DataStax Apache Pulsar(TM) Connector configuration
:navtitle: Update the connector configuration

To update the configuration of an active DataStax Apache Pulsar(TM) Connector sink, use the `pulsar-admin sinks update` command:

[source,bash]
----
pulsar-admin sinks update --name sink_name [options]
----

Replace the following:

* `sink-name`: The name of the sink associated with your DataStax Pulsar Connector.
* `[options]`: Apache Pulsar configuration options to update.
For a complete list, see the Pulsar documentation for `https://pulsar.apache.org/docs/next/io-cli/#update-1[pulsar-admin sinks update]`.

For example, the following command updates a sink named `dse-sink-kv` by setting the parallelism factor to 3:

[source,bash]
----
bin/pulsar-admin sinks update --name dse-sink-kv --parallelism 3
----

To verify the updated configuration, use `xref:ROOT:opsPulsarDisplayConfig.adoc[pulsar-admin sinks get]`.
