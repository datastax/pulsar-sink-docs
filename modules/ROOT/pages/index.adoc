= About the {product}

{product} is open-source software (OSS) installed in the {pulsar-reg} IO framework, and synchronizes records from a {pulsar-short} topic with table rows in {dse}, {astra-db}, and {cass-reg} databases.

Key features include:

* Flexibility in mapping {pulsar-short} messages to {dse-short} and {cass-short} tables.
* Enterprise grade security support including built-in SSL, and LDAP integration.
* Consumes all {pulsar-short} primitives including primitives, JSON and Avro formats.
* Flexible time/date formatting.
* Configurable consistency level.
* Row-level Time-to-Live (TTL).
* Distributed mode, high availability (HA) support.
* Standalone mode support for development.

To get started, see xref:ROOT:pulsarInstall.adoc[] and xref:ROOT:pulsarQuickStart.adoc[].
